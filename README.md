# vue-exp
Маленький учебный проект по работе с фронтендом, на основе уроков с канала Ulbi TV.
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
